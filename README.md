# Neural Networks: Zero to Hero — Build-Along

Working through Andrej Karpathy's "Neural Networks: Zero to Hero" series,
rebuilding each stage from scratch in my own notebooks with my own
comments — not forked/copied code.

## Goal
Understand what's actually happening inside a neural network — 
backpropagation, gradient descent, and eventually a GPT-style transformer 
— from first principles, before relying on frameworks that abstract it away.

## Progress
- [ ] 01: Micrograd — building a tiny autograd engine from scratch
- [ ] 02: Makemore — bigram language model
- [ ] 03: MLP language model
- [ ] 04: Backprop deep dive
- [ ] 05: WaveNet
- [ ] 06: GPT from scratch

## Notes
Each folder contains my own notebook for that stage, with comments 
explaining my understanding as I build — not just following along silently.
